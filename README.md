# phillib

A Lean 4 library for the formalization and analysis of philosophical arguments.

## Mission

Formal logic is a powerful tool for philosophy, but its implementation in proof assistants is often tailored for mathematicians and computer scientists. This creates a barrier for philosophers who want to apply formal methods to their work without getting lost in technical complexities.

**phillib** aims to bridge this gap.

Our mission is to provide a high-level, intuitive, and accessible framework within Lean for philosophers to:

- **Formalize** arguments from philosophical texts in a clear and readable way.
- **Analyze** the logical structure of these arguments.
- **Identify** hidden premises and dependencies.
- **Explore** the consequences of different logical systems (modal, deontic, epistemic, etc.).

This project is not just about implementing logics; it's about building a user-friendly tool that empowers philosophical inquiry.
